# BOJ 12865 - 평범한 배낭

## 문제 요약

N개의 물건이 있고, 각 물건은 무게 `W`와 가치 `V`를 가진다.  
최대 무게 `K`를 넘지 않게 물건을 골라 **가치의 합이 최대**가 되도록 한다.

- 입력 제한: `1 ≤ N ≤ 100`, `1 ≤ K ≤ 100,000`

---

## 풀이 아이디어 (DP 테이블 갱신)

“이전 물건까지의 최적해”를 이용해 “현재 물건을 고려한 최적해”를 갱신한다.

### DP 정의

`dp[i][j]` = **1~i번째 물건까지** 고려했을 때, **무게 한도 j**에서 얻을 수 있는 **최대 가치**

### 갱신(Transition)

1. **현재 물건을 처음 넣은 경우**

- `dp[i][W] = V`

2. **그대로 받는 경우(현재 물건을 안 넣음)**

- 조건: `dp[i-1][j] > dp[i][j]`
- `dp[i][j] = dp[i-1][j]`

3. **현재 물건을 추가해도 한계 무게를 넘지 않는 경우(현재 물건을 넣음)**

- 조건: `j + W <= K && dp[i-1][j] + V > dp[i][j+W]`
- `dp[i][j+W] = dp[i-1][j]+V`
