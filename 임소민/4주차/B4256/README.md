# BOJ 4256 - 트리

## 📌 문제 요약

- 전위 순회(preorder)와 중위 순회(inorder) 결과가 주어진다.
- 이를 이용해 후위 순회(postorder) 결과를 구하는 문제.
- 항상 두 순회 결과로 유일한 이진 트리가 만들어진다.

---

## 💡 풀이 전략

### 🔥 분할 정복 (Divide & Conquer)

이 문제의 핵심은 **루트(root)를 기준으로 나누는 분할 정복**이다.

### 1️⃣ 루트 찾기

- 전위 순회에서 첫 번째 값은 현재 서브트리의 루트이다.

preorder: root → left → right

---

### 2️⃣ 루트를 기준으로 분할

- 중위 순회에서 루트의 위치를 찾는다.
- 그 위치를 기준으로 좌/우 서브트리를 나눈다.

inorder: left \| root \| right

→ 루트 기준으로 정확히 분할 가능

---

### 3️⃣ 재귀적으로 해결

- 오른쪽 서브트리
- 왼쪽 서브트리

를 같은 방식으로 다시 처리한다.

---

### 4️⃣ 후위 순회 구조

postorder: left → right → root

현재 풀이는 루트를 먼저 저장한 뒤 `insert(0, ...)`을 사용하여\
결과적으로 후위 순회 형태가 되도록 구성하였다.

---

## ⏱ 시간 복잡도

- inorder에서 루트 위치 탐색: O(n)
- 최악의 경우 O(n²)
- n ≤ 1000 이므로 통과 가능

---

## 🎯 정리

- 전위 + 중위 → 트리 복원
- 분할 기준은 항상 **루트**
- 전형적인 **분할 정복 문제**
