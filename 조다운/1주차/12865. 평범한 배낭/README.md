### 문제 해석

- 물건 N개 1 ≤ N ≤ 100
- 각 물건: 무게 w, 가치 v
- 배낭은 최대 무게 k 까지 담을 수 있음
- 무게 합은 k 를 넘을 수 없음
- 최대한 높은 가치의 배낭을 만들기!!

### 고려 조건

- 완탐은 안 됨
- dp! 이전 문제까지의 최적해를 바탕으로 현재 문제의 최적해를 구하게 된다.

### 엌 케 풀음 고민 중

- 지금까지 몇 개 물건을 고려했고, 현재 배낭 무게가 `w` 일 때 얻을 수 있는 최대 가치
- dp[j] 가 무게 j까지 써서 얻을 수 있는 최대 가치로 두었을 때…!

### 풀이 과정

먼저 `dp[j]` 는 물건을 담지 않았을 때, 넣었을 때로 나뉨!

물건 하나 `(w,v)` 에 도달했을 때 `dp[j]` 에 이 물건을 넣으면 이 물건이 무게 `w` 를 차지하니까 남은 무게는 `dp[j-w]` 그러므로 그 상태에서 얻을 수 있는 최대 가치는 `dp[j-w]` 임 그리고 거기다가 현재 물건의 가치 `v`를 더함

dp[j]는 j 까지 써서 얻는 최대 가치이고, dp[j-w] + v는 현재 물건을 넣었을 떄의 경우!

앞에서부터 갱신한다면, 현재 물건을 한 번만 써야 하는데 갱신된 값을 다시 참조하게 되는 경우가 생겨서 같은 물건의 dp 값을 2번 쓰게 되는 일이 생긴다.
