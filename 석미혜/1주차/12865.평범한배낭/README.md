## 📝 문제 요약

준서는 여행에 필요하다고 생각하는 **N개의 물건** 중에서

**버틸 수 있는 최대 무게 K를 넘지 않는 선에서**

물건들의 **가치(V)의 합이 최대**가 되도록 선택하려 한다.

각 물건은 **한 번만 선택 가능**하며,

배낭에 담을 수 있는 **최대 가치**를 구하는 문제이다.

---
## ✅ 풀이 과정

### 1️⃣ 문제 분석

- 물건마다 무게(W)와 가치(V)가 주어짐
- 각 물건은 **0번 또는 1번만 선택 가능**
- 최대 무게 제한 K를 넘지 않도록 조합해야 함

---

### 2️⃣ 해결 전략 수립

- 모든 조합을 탐색하면 경우의 수가 너무 많아 비효율적
- **DP를 이용해 부분 문제의 최적해를 누적**
- 현재 물건을:
    - **선택하지 않는 경우** → 기존 `dp[w]`
    - **선택하는 경우** → `dp[w - W] + V`
- 두 경우 중 더 큰 값을 `dp[w]`에 저장

----

### 3️⃣ 코드 구현

- DP 배열을 `K + 1` 크기로 생성
- 각 물건마다 무게를 **뒤에서부터** 순회하며 dp 갱신
