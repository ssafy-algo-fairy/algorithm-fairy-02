## 📝 문제 요약

회전 초밥 벨트에 놓인 초밥들 중에서 **연속된 k개의 초밥**을 선택해 먹을 수 있다.

또한 **쿠폰 번호에 해당하는 초밥 1개를 추가로 먹을 수 있는 기회**가 주어진다.

이때, 손님이 **먹을 수 있는 서로 다른 초밥 가짓수의 최댓값**을 구하는 문제이다.

## 💡 접근 방법

- **알고리즘 / 자료구조:** 슬라이딩 윈도우, 배열
- **핵심 아이디어:**
    - 초밥 종류별 개수를 저장하는 배열을 사용한다.
    - 처음 k개의 초밥에 대해 각 종류의 등장 횟수와 현재 종류 수를 계산한다.
    - 이후 윈도우를 한 칸씩 이동시키며
        - 빠지는 초밥은 개수를 감소시키고
        - 새로 들어오는 초밥은 개수를 증가시킨다.
    - 매 이동마다 **쿠폰 초밥이 포함되어 있는지 확인하여 가짓수 최댓값을 갱신**한다.

## ✅ 풀이 과정

1. **문제 분석**
    - 원형 구조이므로 인덱스를 `% n`으로 처리해야 함
    - 쿠폰 초밥은 기존에 포함되지 않았을 때만 가짓수에 +1
2. **해결 전략 수립**
    - 길이 k의 슬라이딩 윈도우를 유지하며 초밥 종류 수를 관리
    - 초밥 종류별 개수를 저장하는 배열 사용
3. **코드 구현**
    - 초기 k개 구간의 초밥 상태 설정
    - 슬라이딩 윈도우로 한 칸씩 이동하며 상태 갱신
    - 매 단계마다 최댓값 비교
4. **테스트 및 검증**
    - 쿠폰 초밥이 이미 포함된 경우 / 포함되지 않은 경우 모두 확인
    - 원형 인덱스 처리 정상 동작 여부 검증
