## 📝 문제 요약

하늘에서 별똥별이 빗발친다.

- 하늘에서 떨어지는 별똥별의 위치가 주어질때 L*L 크기의 트램펄린을 배치하여 별똥별이 최소한으로 땅으로 떨어지도록 


## 💡 접근 방법
    
- **핵심 아이디어:**
    
  별동별이 트램펄린 모서리에 포함되어있어야 최대한 많은 면적으로 별똥별 커버가 가능하다.
  백준 14658번(하늘에서 별똥별이 빗발친다) 문제를
  요청한 형식에 맞춰 정리해줄게.

## ✅ 풀이 과정
### 1. 문제 분석

  N × M 크기의 평면 위에 K개의 별이 존재한다.
  
  한 변의 길이가 L인 정사각형 트램펄린을 한 번 설치할 수 있다.
  
  트램펄린 안에 포함되는 별은 튕겨나가지 않는다.
  
  목표는 튀어 나가는 별의 개수를 최소화하는 것이다.

 **🔎 핵심 관찰**

트램펄린을 아무 위치에나 둘 수 있지만,

최적으로 많이 담는 위치는 항상

세로 변이 어떤 별의 x좌표에 맞거나, 
가로 변이 어떤 별의 y좌표에 맞는 위치로 이동시킬 수 있다.

즉, 무한한 좌표를 탐색할 필요 없이
별 좌표 기준으로만 후보를 만들면 된다.

### 2. 해결 전략 수립
   **🔥 핵심 아이디어**

트램펄린의 왼쪽 아래 좌표를

(stars[i][0], stars[j][1])


조합으로 전부 만들어본다.

각 후보 위치마다

범위:

x: x1 ~ x1 + L
y: y1 ~ y1 + L


해당 범위 안에 들어오는 별 개수를 센다.

전체 별 K에서

(사각형 안에 들어간 별 개수)


를 빼면 튕겨 나가는 별 개수.

이 값을 최소로 갱신한다.
